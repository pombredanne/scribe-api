Mixin: Scribe.Mixins.Triggerable

Defined in: src/scribe/mixins/triggerable.coffee
Included in: Scribe.Menu Scribe.Window

Overview

Allows events to be registered and triggered off the object.

Method Summary

  • ~ (void) on(eventName, callback) Register a callback to run when the given event occurs
  • ~ (void) off(eventName, callback) Deregister a specific callback or all callbacks for an event
  • ~ (void) trigger(eventName) Synchronously run the callbacks that have been associated with eventName (registered by calling on()).

Method Details

~ (void) on(eventName, callback)

Register a callback to run when the given event occurs

Parameters:

  • eventName ( String ) — the name of the event
  • callback ( Function ) — the code to run when the event fires

~ (void) off(eventName, callback)

Deregister a specific callback or all callbacks for an event

Parameters:

  • eventName ( String ) — the name of the event
  • callback ( Function ) — when specified, this specific callback is removed from the list of callbacks. When omitted, all callbacks registered for {eventName} are removed.

~ (void) trigger(eventName)

Synchronously run the callbacks that have been associated with eventName (registered by calling on()).

Parameters:

  • eventName ( String ) — the name of the event
