Method Details

~ (void) set(keyOrObj, val)

A helper method for changing one or more instance properties.

Examples:

using the Object syntax: 

car.set({ speed: 0.5, acceleration: 1.2 })

using the key, val syntax: 

car.set('speed', 0.5);
car.set('acceleration', 1.2);

Parameters:

  • keyOrObj ( String, Object ) — either an Object literal containing key, value pairs to set on the instance, or a String that contains the key to store val under.
  • val ( Object ) — the value to set, required when keyOrObj is a String.

Throws:

  • ( Error ) — when incorrect arguments are given
