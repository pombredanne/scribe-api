Class: Scribe.DockIcon

Defined in: src/scribe/dock_icon.coffee

Overview

An icon in the system dock or tray.

Instance Properties

(String, null) url
a URL to display in the DockIcon; when this is null, the icon defaults to the URL specified in the Scribe manifest. Setting this property updates the icon immediately.
(String, null) badge
the contents of the badge label on the DockIcon; when this is null, the badge is hidden.
(Scribe.Menu, null) contextMenu
the right-click context menu associated with this DockIcon.

Instance Methods

  • # (void) bounce() Bounces the app icon in the dock, if possible.
