Class: Scribe.Plugin

Defined in: src/scribe/plugin.coffee

Overview

An interface to a native plugin.

Examples:


var adLibrary = Scribe.Plugin.load('GenericAds');
adLibrary.call('init', 'AFB15610CCCEF1101');
adLibrary.call('reportEvent', 'launch');
