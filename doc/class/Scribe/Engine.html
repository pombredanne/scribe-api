Class: Scribe.Engine

Defined in: src/scribe/engine.coffee

Overview

An Engine tracks a single Javascript execution context and provides a few runtime fundamentals like eval and parent context tracking.

Class Properties

(Scribe.Engine) current
A reference to the Engine that is currently running the caller code.

Instance Properties

(Scribe.Engine) parent
A reference to the Engine that created this Engine.
(Object) nativeObject
A reference to the native Engine object. The exact type and API of this object will be platform-dependent.
(Object) jsGlobal
A reference to the Javascript global object from the embedded WebView inside this Window.

Instance Methods

  • # (Object) eval(jsCode) Evaluates the given jsCode in the context of the Engine
  • # (void) repl() Starts a small Javascript REPL (with readline bindings!) on stdin and blocks until the user runs exit().
