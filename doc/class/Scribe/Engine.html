Class: Scribe.Engine

Defined in: src/scribe/engine.coffee

Overview

The Engine tracks the Javascript execution context and provides a few runtime fundamentals like eval and parent context tracking.

Class Properties

(Scribe.Engine) current
A reference to the Engine that is currently running the caller code.
(Scribe.Engine) parent
A reference to the Engine that created the current Engine.

Instance Properties

(Scribe.Engine) parent
A reference to the Engine that created this Engine.
(Object) nativeObject
A reference to the native Engine object. The exact type and API of this object will be platform-dependent.

Instance Methods

  • # (void) eval(jsCode) Evaluates the given jsCode in the context of the Engine.
