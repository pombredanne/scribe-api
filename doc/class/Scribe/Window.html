Class: Scribe.Window

Defined in: src/scribe/window.coffee
Includes: Scribe.Mixins.Triggerable

Overview

Manipulates a native window that contains an embedded browser. The embedded browser can load an arbitrary URL. Code running in the embedded browser can access the Window instance through the Scribe.Window.current property.

Class Properties

(Scribe.Window) current

A reference to the Window that is currently running the caller code. If the caller code is not associated with a Window (e.g. the entrypoint script), null is returned.

Instance Properties

(Boolean) fullscreen
the Window covers the entire screen
(String) title
the title of the Window. If chrome is true, this String will be displayed above the Window.
(Boolean) chrome
display the window chrome (the titlebar)
(Boolean) closable
the Window can be closed by the user
(Boolean) resizable
the Window can be resized by the user
(Boolean) visible
the Window is currently visible on screen
(Number) left
the offset (in pixels) from the left of the screen
(Number) top
the offset (in pixels) from the top of the screen
(Number) width
the absolute width (in pixels) of the window
(Number) height
the absolute height (in pixels) of the window
(Number) alpha
the alpha percentage (between 0 and 1)
(Boolean) topmost
this window is pinned above all others
(Boolean) sameOriginPolicy
the embedded browser enforces Same Origin Policy
(Object) nativeWindowObject
A reference to the native window object. The exact type and API of this object will be platform-dependent.
(Scribe.Engine) engine
the Scribe.Engine instance that has been injected into the currently-loaded Javascript context.
