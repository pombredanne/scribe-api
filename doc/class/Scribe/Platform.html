Class: Scribe.Platform

Defined in: src/scribe/platform.coffee

Overview

A system that is hosting the Scribe runtime

Class Properties

(Array<String>) SUPPORTED
A list of supported platforms:
  • osx
  • ios
(Scribe.Platform) current
A reference to the host's Platform

Instance Properties

(String) name
The operating system name. See the SUPPORTED class variable.
(String) version
The operating system version.

Instance Methods

  • # (Boolean) is(name) Determine whether name matches the Platform

    For accepted values see the SUPPORTED class variable.
  • # (void) exec(cmd, opts = {}) Runs a shell command in the background.
  • # (Object) execSync(cmd, opts = {}) Runs a shell command, blocks until completion, and returns the output.

Instance Method Details

# (Boolean) is(name)

Determine whether name matches the Platform

For accepted values see the SUPPORTED class variable.

Examples:

if Scribe.Platform.current.is('windows') {
   console.log('Nope!');
   Scribe.app.exit(1);
}

Parameters:

  • name ( String ) — the name to match

Returns:

  • ( Boolean ) — whether name matches the Platform

# (void) exec(cmd, opts = {})

Runs a shell command in the background.

Parameters:

  • cmd ( String ) — the shell command to run
  • opts ( Object ) — the options hash

Options Hash: (opts):

  • 'done' ( Function ) — the callback function that accepts the parameters stdout, stderr

# (Object) execSync(cmd, opts = {})

Runs a shell command, blocks until completion, and returns the output.

Parameters:

  • cmd ( String ) — the shell command to run
  • opts ( Object ) — the options hash

Options Hash: (opts):

  • 'done' ( Function ) — the callback function that accepts the parameters stdout, stderr

Returns:

  • ( Object ) — an Object with stdout and stderr keys that point to Strings containing the buffer contents
